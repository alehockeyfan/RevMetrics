Revenue Analysis

🇬🇧 Quick Guide

  • Language Selector: Use the dropdown in the top-right corner to switch between English and Italian at any time.
  • File Requirement: Upload an Excel file containing at least the following column headers: "Ragione Sociale", "Imponibile", and "Data documento".
  • Privacy: Your data is processed entirely inside your web browser. No files are uploaded to any external server.
  • Bayesian Score: The reliability rating measures client value and consistency (0-100), decaying if no invoices are registered for a long time.
  • Merge Clients: Use this tool if a client has changed their company name. Combining their historical records will ensure you get precise, unified statistics.

🇮🇹 Guida Rapida

  • Scelta Lingua: Usa il menu a tendina in alto a destra per cambiare la lingua tra Italiano e Inglese in qualsiasi momento.
  • Requisiti File: Carica un file Excel che contenga almeno le colonne denominate: "Ragione Sociale", "Imponibile" e "Data documento".
  • Privacy: I tuoi dati sono elaborati interamente all'interno del browser. Nessun dato viene inviato a server esterni.
  • Punteggio Bayesiano: L'indice di affidabilità (0-100) valuta la costanza del cliente, calando se non vengono registrate fatture da molto tempo.
  • Unisci Clienti (Merge): Utilizza questo strumento se un cliente ha cambiato ragione sociale nel tempo. Unendo i record storici otterrai statistiche precise e unificate.
